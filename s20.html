一、软件资讯

 

▼ 国内网站 简介
🌐 GitHub 全球最大开源托管平台，开发者们“殿堂般”。
🌐 52Pojie GitHub 和 52Pojie 合称“北乔峰南慕容”，国内活跃度最高的软件交流社区。
🌐 小众软件 老牌软件分享网站，介绍各种新鲜软件，还新开了一个“发现分享”频道。
🌐 异次元软件 与“小众软件”类似，但更新强度不如以前。
🌐 Topbook 可以说是“视频版小众软件”，信噪音比不高，同样的内容需要的时间更多。
🌐 少数派 软件界的“小红书”，文章更多的是“经验分享”，但苹果用户占比太高。
🌐 IT 之家 专业的 IT 新闻网站，有一个“软件”频道，软件界的大新闻，这里都有。
▼ 国外网站 简介
🌐 Product Hunt 国外最火的软件应用资讯网站，能发掘很多前沿新产品。
🌐 BetaList 与 Product Hunt 类似，但提供的都是新鲜前沿的应用。
🌐 Slant 提供各种应用的排名测评，有正反观点，比如“最好的浏览器是什么”？
▼ 相似软件 简介
🌐 Product Hunt 网外最火的软件资讯网站，右下角的”View all alternatives”就是相似软件。
🌐 AlternativeTo 老牌网站，支持 Windows、macOS、linux、网页、iOS、安卓。
🌐 Altapps.net 支持多语言，支持多平台，资源也比较丰富。
🌐 ProgSoft 支持桌面软件，热度也很高。
🌐 Alternative.me 支持桌面软件，资讯相对较老。
🌐 Alternatives.app 支持桌面软件，资讯相对较老。
🌐 Similarsites 发明相似网站。
🌐 Similarweb 发明相似网站和手机应用。
🌐 开源替代方案 200 多个开源替代品，可替代企业日常运营所需的工具。
🌐 MrFreeTools 查找收费应用的免费方案。
