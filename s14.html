1、国内问答

 

▼ 国外网站 简介
🌐 知乎 国内最受欢迎的问答网站。
🌐 百度知道 曾经国内最受欢迎的问答网站。
🌐 百度经验 更加详细具体的“百度知道”。
🌐 知识星球 付费问答社区，付费社群。
🌐 问答库 考试题库网站，不仅是大学生之友，而且还非常适合需要考证的人。
🌐 考试酷 考试题库网站，免费的电子作业与在线考试系统云平台。
🌐 雨露学习互助 考试题库网站，主要向面初高中学生。
🌐 SegmentFault 国内类似 StackOverflow 的技术问答网站。
🌐 搜外问答 专注站长 SEO 交流的平台。
🌐 天天问 产品经理互助问答社区。

 

2、国外问答

 

▼ 国外网站 简介
🌐 Quora 国外最受欢迎的问答网站。
🌐 Answers 国外版“百度知道”。
🌐 AskReddit 美国版“贴吧”，新出 AskReddit 又类似国内的“知乎”。现在风头正盛。
🌐 WikiHow 全球最实用的百科指南、生活指南、万事指南。
🌐 Wolfram Alpha 知识搜索网站，能根据问题就可以直接给出答案的网站。
🌐 Stack Exchange 多领域问答网站，主打“专业人士”的知识交流平台。
🌐 Stack Overflow StackExchange 的子网站，但单独出来了，主攻“技术编程”类内容。
🌐 Diffen 输入任意两个内容，都可以进行全方位的对比。

 

3、社交网站

 

▼ 社交网站 简介
🌐 微博 随时随地发现“新鲜事”。
🌐 微信 微信搜索引擎，包括公众号、视频号、服务号。
🌐 豆瓣 图书、电影、音乐的推荐和评论，以及城市独特的文化生活。
🌐 知乎 国内知名问答社区，连接各行各业用户。
🌐 果壳 提供有趣，贴近生活的泛科技主题网站。
🌐 贴吧 国内互联网里的“卢浮宫”，涉及方方面面。
🌐 Twitter 看看世界上的“新鲜事”。
🌐 Facebook 美国最大社交网站。
🌐 Instagram 图片视频社交网站。
🌐 Medium 英文版简书。
🌐 Quora 英文版知乎。
🌐 Reddit 美国版贴吧。
